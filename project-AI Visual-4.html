01

프로젝트 개요

촬영 없이도 활용 가능한 패션 콘텐츠를 제작하기 위해 AI 모델 생성과 상품 이미지를 결합한 룩북 이미지를 기획하고 제작했습니다. 단순 합성에 그치지 않고, 실제 쇼핑몰에서 활용 가능한 수준의 구도, 스타일링, 컷 구성을 고려하여 작업을 진행하였고 모델의 일관성을 유지하는것을 중요한 목표로 생각했습니다. 모델 생성부터 착장 이미지 제작, 룩북 컷 구성까지 전 과정을 직접 기획하여 작업했습니다.

03

주요작업

1. AI 기반 패션 모델 이미지 생성 (Midjourney 활용) 2. 제품 이미지(자켓, 스커트, 가방) 선정 및 스타일링 기획 3. 모델과 제품의 자연스러운 착장 합성 작업 4. 쇼핑몰 활용을 고려한 룩북 컷 구성 기획 (전신 / 워킹 / 측면 / 후면) 5. 전체 이미지 톤 & 무드 디렉팅 (미니멀 / 뉴트럴 톤)

04

결과물 특징

1. 실제 촬영 없이 제작된 룩북 이미지 구성 2. 브랜드 촬영과 유사한 자연스러운 조명과 톤 구현 3. 다양한 포즈와 구도를 통한 활용도 높은 콘텐츠 제작

05

인사이트

촬영 없이도 AI를 활용해 상업용 콘텐츠를 제작할 수 있는 가능성을 확인했습니다. 특히 모델 섭외, 촬영 등의 비용 없이도 일관된 퀄리티의 이미지를 빠르게 제작할 수 있다는 점에서 효율성을 확보할 수 있었습니다. 앞으로는 디테일 표현과 제품 정확도를 더욱 개선하여 실제 커머스 환경에서도 활용 가능한 수준으로 확장할 수 있을 것 같습니다
