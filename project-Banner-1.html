프로젝트 개요

토리든 ‘저분자 히알루론 수분 세럼’을 기반으로 제품 기능을 직관적으로 전달하는 배너 세트 기획 프로젝트입니다. 제품 컷을 중심으로 AI 이미지 비주얼을 생성·합성하여 기능 강조형, 메시지 중심형, 프로모션 중심형, SNS 세로형 등 목적별 배너 구조를 설계했습니다. 동일 제품을 다양한 마케팅 상황에 대응할 수 있도록 구매 전환을 고려한 레이아웃과 카피 구조를 전략적으로 구성했습니다.

프로젝트 완성

03

주요작업

1.배너 전략 구조 설계 - 기능 강조형 / 카피 중심형 / 혜택 강조형 / SNS 세로형 구조 기획 - 전환 목적에 따른 메시지 배치 설계 2. AI 기반 비주얼 생성 및 합성 - 물·수분 텍스처 이미지 생성 - 제품 합성 및 각도·빛 반사 보정 - 제품 중심 시각 집중도 설계 3. 카피 및 메시지 구조 정리 - ‘저분자 히알루론 수분 세럼’ 핵심 기능 강조 - 속보습·집중보습 메시지 시각화 - 프로모션 요소(1+1, 리필기획) 시각 임팩트 강화 4. 모바일 최적화 레이아웃 구성 - 세로형 SNS 광고 대응 구조 설계 - 가독성 중심 타이포 계층 구조 정리

05

인사이트

제품 중심 배너는 감성보다 기능 전달과 시선 흐름 설계가 중요하며, 동일 제품이라도 목적에 따라 구조를 달리해야 전환 효율을 높일 수 있다는 점을 확인했습니다.
