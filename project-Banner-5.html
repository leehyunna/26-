01

프로젝트 개요

피그마와 Google Sheets Sync 플러그인을 활용하여 상품 정보를 기반으로 카드뉴스 콘텐츠를 자동 생성할 수 있도록 작업했습니다. 반복적으로 발생하는 상품 콘텐츠 제작 과정에서 데이터 입력만으로 디자인이 반영되도록 하여 작업 효율성과 콘텐츠 일관성을 동시에 개선하는 데 목적을 두었습니다.

주요작업

1. Google Sheets 기반 콘텐츠 데이터 구조 설계 (상품명, 사이즈, 문구 등) 2. 피그마 플러그인을 활용한 데이터 연동 3. 피그마 오토레이아웃 기반 카드뉴스 템플릿 제작 4. 데이터 수정 시 디자인이 자동 반영되는 구조 설계

05

인사이트

이커머스 콘텐츠는 반복 생산되는 특성이 강하기 때문에 디자인 결과물도 중요하지만 반복되는 작업의 경우 효율성을 극대화 시키는 업무 구조를 설계하는 것이 더 중요하다고 판단했습니다. Google Sheets와 피그마를 연동함으로써 단순 반복 작업을 줄이고, 작업 시간 단축과 디자인 일관성을 동시에 확보할 수 있었습니다.
